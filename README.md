# Quality Audit System

Aplikasi berbasis **Flutter + Firebase** untuk mendukung kegiatan **Audit K3 (Keselamatan dan Kesehatan Kerja)**.  
Sistem ini membantu mendeteksi bahaya, menilai risiko, dan menampilkan hasil audit dalam bentuk dashboard interaktif.

---

## Fitur Utama

- Formulir audit produk
- Deteksi otomatis potensi bahaya (hazard)
- Dashboard statistik (Total Audit, Hazard, Risiko Tinggi)
- Grafik tren audit bulanan (Line Chart)
- Data real-time dari **Firebase Firestore**
- Tampilan responsif (desktop & mobile)

---

## Alur Singkat

1. Auditor mengisi data audit.  
2. Data disimpan ke Firestore.  
3. Sistem menghitung risiko otomatis.  
4. Hasil divisualisasikan di Dashboard.
